# Realtime Chat Application

A modern real-time messaging application that enables instant communication between users. Built with WebSocket technology for seamless, low-latency messaging experience.

## Core Features

- **Instant Messaging**: Real-time message delivery across all connected clients
- **Group Chats**: Create and manage group conversations
- **File Sharing**: Share images and documents with automatic compression
- **Emoji Reactions**: React to messages with emojis
- **Typing Indicators**: See when someone is typing
- **Message Status**: Delivered and read receipts
- **Online Status**: See who's currently online
- **Push Notifications**: Browser notifications for new messages

## Technical Implementation

- **Frontend**: React with real-time state management
- **Backend**: Node.js with Socket.io for WebSocket connections
- **Database**: MongoDB for message storage
- **Cache**: Redis for session management
- **File Storage**: Cloudinary for media files
