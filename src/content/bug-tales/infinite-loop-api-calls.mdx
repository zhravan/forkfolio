  3. Response updates state
  4. State change retriggers `useEffect`
  5. Repeat â†’ flood of requests
solution: |
  ```tsx
  // Before
  useEffect(() => {
    fetchUserData(user.id)
  })

  // After
  useEffect(() => {
    fetchUserData(user.id)
  }, [user.id])
  ```
  Additional fixes: request debouncing, circuit breaker, and `AbortController` cancellation.
